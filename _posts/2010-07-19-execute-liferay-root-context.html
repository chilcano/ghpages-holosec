Sometime we need to execute Liferay Portal in a different web context, for example in http://www.intix.info/myportal instead of http://www.intix.info/

Then, in this scenario we have to follow next steps:

  1. Rename <LIFERAY_HOME>/<TOMCAT_HOME>/webapps/ROOT   to <LIFERAY_HOME>/<TOMCAT_HOME>/webapps/myportal
  2. Edit or create portal-ext.properties (<LIFERAY_HOME>/<TOMCAT_HOME>/webapps/myportal/WEB-INF/classes/portal-ext.properties) :

    3. ...
    portal.ctx=/myportal
    ....

  3. Rename <LIFERAY_HOME>/<TOMCAT_HOME>/conf/Catalina/localhost/ROOT.xml TO <LIFERAY_HOME>/<TOMCAT_HOME>/conf/Catalina/localhost/myportal.xml
  4. Restart Liferay and go to  http://www.intix.info/myportal , should be possible to browser around liferay without errors, if you found broken links you must change absolut URL/Path adding to begin "/myportal", for example:  update "http://www.intix.info/image/image_gallery?xyzx" with "http://www.intix.info/myportal/image/image_gallery?xyz"

Bye.
